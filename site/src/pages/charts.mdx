## Available Charts

<div class="charts-grid">

[**`12factor`**](https://github.com/mjpitz/mjpitz/tree/main/charts/12factor)  
Easily deploy an application that conforms to 12factor application patterns.

[**`cognative`**](https://github.com/mjpitz/mjpitz/tree/main/charts/cognative)  
Deploy the [Cognative][] business intelligence and operations platform.

[**`drone`**](https://github.com/mjpitz/mjpitz/tree/main/charts/drone)  
Deploys a [Drone][] server and optionally configure a `digitalocean` or `kubernetes` runner.

[**`gitea`**](https://github.com/mjpitz/mjpitz/tree/main/charts/gitea)  
Git with a cup of tea. [Gitea][] is a painless, self-hosted Git Service.

[**`litestream`**](https://github.com/mjpitz/mjpitz/tree/main/charts/litestream)  
Easily add a [Litestream][] sidecar to any `Deployment`, `DaemonSet`, or `StatefulSet`.

[**`maddy`**](https://github.com/mjpitz/mjpitz/tree/main/charts/maddy)  
Deploys a single-tenant Maddy email server.

[**`raw`**](https://github.com/mjpitz/mjpitz/tree/main/charts/raw)  
Deploys raw Kubernetes resources to the cluster.

[**`redis`**](https://github.com/mjpitz/mjpitz/tree/main/charts/redis)  
Deploys an inconsistent Redis cluster, intended to be fronted by an [Envoy][] sidecar.

[**`redis-queue`**](https://github.com/mjpitz/mjpitz/tree/main/charts/redis-queue)  
Deploy a single durable Redis instance, intended to be used as a work queue.

[**`redis-raft`**](https://github.com/mjpitz/mjpitz/tree/main/charts/redis-raft)  
Deploy a consistent, partition-tolerant Redis cluster backed by the Raft consensus protocol.

[**`registry`**](https://github.com/mjpitz/mjpitz/tree/main/charts/registry)  
Deploys a replicated container registry, with optional caching provided by `redis`.

[**`renovate`**](https://github.com/mjpitz/mjpitz/tree/main/charts/renovate)  
Deploys Renovate for automatically managing your software dependencies.

[**`storj`**](https://github.com/mjpitz/mjpitz/tree/main/charts/storj)  
Easily add a Storj S3 Gateway as a sidecar to any `Deployment`, `DaemonSet`, or `StatefulSet`.

</div>
